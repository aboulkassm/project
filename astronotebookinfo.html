AstroNotebook

About the App

  • AstroNotebook is an innovative web application that allows users to create, save, and delete notes in a visually appealing and interactive environment.
  • Developed using HTML, CSS, and JavaScript, AstronoteApp offers a seamless note-taking experience with CSS animations and a dynamic live background.
  • Users can easily create new notes, add content, and apply various formatting options to customize their notes according to their preferences.
  • The app leverages CSS animations to provide delightful transitions and visual feedback when creating, editing, or deleting notes, enhancing the user experience.
  • With the live background feature, AstronoteApp offers a unique touch by dynamically changing the background, displaying captivating images or animations related to the astral theme.
