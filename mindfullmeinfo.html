MindfulMe App

About the App

  • Collaborated with a university group to develop an Android app for effective well-being.
  • Utilized Java programming language and a SQL database for data collection and storage.
  • Features included user input for well-being metrics like mood, energy levels, and sleep quality.
  • Provided personalized exercise suggestions and mindfulness techniques based on user data.
  • Integrated concentration measurement using device sensors to assess focus levels during tasks.
  • Displayed visually appealing charts and statistics to track progress and identify patterns.
  • Ensured privacy and secure handling of personal data.
  • Leveraged Git version control and GitHub for seamless collaboration within the university group.
